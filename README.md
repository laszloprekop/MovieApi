# MovieApi

A movie REST API. ASP.NET Core Web API with EF Core.

Learning project (Lexicon LTU, Övning 3).

## Stack

- .NET 10
- EF Core (SQL Server / LocalDB)

## Run

```bash
dotnet run --project MovieApi
```

## Migrations

```bash
dotnet ef migrations add <Name>
dotnet ef database update
```
